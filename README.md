# Apprentice Management System

A web-based Apprentice Management System developed using PHP, MySQL, and Bootstrap. The system provides role-based access control and enables efficient management of apprentice information, sections, trades, and administrative activities.

## Features

### Authentication & Authorization
- Secure login system
- Role-based access control
- Session management

### Apprentice Management
- Add apprentice details
- Update records
- View apprentice information
- Manage apprentice categories

### Section Management
- Section Head Dashboard
- Section-wise apprentice allocation
- Trade-wise filtering

### Trade Management
Supported Trades:
- COPA
- Electrician
- Sheet Metal Worker
- Welder
- Mechanic Diesel
- MMV
- Tractor Mechanic
- Machinist
- Fitter
- Turner
- Plumber
- Draughtsman
- Carpenter
- Surveyor
- Machinist (Grinder)
- Painter
- Stenographer
- ICTSM

## Installation

1. Clone the repository

```bash
git clone https://github.com/yourusername/Apprentice-Management-System.git
```

2. Move the project to your XAMPP/WAMP htdocs folder.

3. Create a MySQL database.

4. Import the SQL file.

5. Configure database credentials.

6. Start Apache and MySQL.

7. Open:

```text
http://localhost/Apprentice-Management-System
```
